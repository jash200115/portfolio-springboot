My Background

Where DevOps Meets Backend Engineering.

Hi, I’m Jash Roy, a Software Development Engineer I at Mastercard. I began my journey as a DevOps engineer, where I focused on building reliable, automated, and observable systems at scale. My work included creating advanced Splunk dashboards, automating deployments using custom Python modules integrated with Stash, and collaborating across a wide range of tools such as Artifactory, Chef, Packer, Xray, Bitbucket, and WAF.

One of my key contributions was designing a production-ready Packer template for Ubuntu 22.10 with Azure Arc integration, enabling consistent and secure infrastructure provisioning. This experience gave me a strong foundation in infrastructure automation, CI/CD, and system reliability.

Recently, I’ve expanded my role into Java backend development, building applications using Spring Boot with IntelliJ IDEA. This transition allows me to combine my DevOps background with hands-on software engineering—bridging the gap between infrastructure and application development to deliver scalable, automated, and observable systems end-to-end.
